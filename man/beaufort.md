beaufort(1) -- Command line Beaufort Cipher utility
=================================

## SYNOPSIS

`beaufort` [-hV] [options]

## OPTIONS

  -V, --version
      output program version

  -h, --help
      output help information

  --encrypt
      encrypt stdin stream

  --decrypt
      decrypt stdin stream

  --key=[key]
      cipher key (required)

  --alphabet=[alpha]
    cipher tableau alphabet (Default: 'abcdefghijklmnopqrstuvwxyz')
